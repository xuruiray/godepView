## godepView

#### 功能：
根据 go 项目中自身包依赖绘制依赖关系图，并通过 html 页面形式展示


#### 如何安装：
```
go get github.com/xuruiray/godepView
go install github.com/xuruiray/godepView
```

#### 如何使用：
```
godepView {package path}
```
